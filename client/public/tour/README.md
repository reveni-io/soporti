# Tour screenshots

Optional screenshots shown in the "What can I ask?" tour modal (`TourModal.jsx`).
Steps hide their image automatically when the file is missing, so the tour works
without them.

Expected files (PNG, ideally ~1000px wide, light theme):

- `sources.png` — the sidebar "Sources" section, showing YOLO (auto) plus some
  integrations and repos selected.
- `profiles.png` — the sidebar "Profile" toggle (Support / Tech).
- `data-answer.png` — a chat answer to a data question, ideally one that renders
  a chart or a table.
